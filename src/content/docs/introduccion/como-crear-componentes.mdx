# Cómo Crear Componentes

Glory permite crear componentes reutilizables siguiendo patrones consistentes. Todos los componentes van en `Glory/src/Components/`.

## Estructura básica

```
Glory/src/Components/
├── TuComponente.php          # Clase principal
└── TuComponente/
    └── assets/              # CSS y JS opcionales
```

## Crear componente básico

**Uso mínimo** - Componente simple:
```php
<?php
namespace Glory\Components;

class MiComponente
{
    public static function render($args = [])
    {
        $texto = $args['texto'] ?? 'Hola Mundo';
        return "<div class='mi-componente'>{$texto}</div>";
    }

    public static function print($args = [])
    {
        echo self::render($args);
    }
}
```

**Uso avanzado** - Con validación y configuración:
```php
<?php
class MiComponenteAvanzado
{
    public static function render($args = [])
    {
        $defaults = ['texto' => 'Hola', 'clase' => 'componente'];
        $config = array_merge($defaults, $args);

        // Validar
        if (empty($config['texto'])) {
            return '<div class="error">Texto requerido</div>';
        }

        $clase = esc_attr($config['clase']);
        $texto = esc_html($config['texto']);

        return "<div class='{$clase}'>{$texto}</div>";
    }
}
```

¡Crea componentes siguiendo estos patrones simples!
