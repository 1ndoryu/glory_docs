## Features disponibles en Glory

Lista de funcionalidades que puedes activar/desactivar:

## Control básico

```php
use Glory\Core\GloryFeatures;

// Activar feature
GloryFeatures::enable('miFeature');

// Desactivar feature
GloryFeatures::disable('miFeature');

// Verificar si está activa
if (GloryFeatures::isActive('miFeature')) {
    // Código aquí
}
```

## Features principales

- **Managers**: menu, assetManager, pageManager, postTypeManager, etc.
- **UI**: modales, alertas, pestanas, scheduler, etc.
- **Services**: cssCritico, gloryAjax, gloryRealtime, etc.
- **Renderers**: contentRender, termRender, logoRenderer
- **Theme**: titleTag, postThumbnails

Usa `GloryFeatures::isActive()` para controlar assets y funcionalidades.
