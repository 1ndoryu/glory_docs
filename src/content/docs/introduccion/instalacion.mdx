# Instalación de Glory Framework

## Requisitos
- WordPress 5.0+
- PHP 7.4+
- Composer
- Git

## Pasos de instalación

**Uso mínimo** - Instalación básica:
```bash
# 1. Clona el tema base
cd wp-content/themes
git clone https://github.com/1ndoryu/glorytemplate.git

# 2. Instala dependencias
cd glorytemplate
composer install

# 3. Clona Glory
git clone https://github.com/1ndoryu/glory.git

# 4. Crea archivo .env
echo "LOCAL=TRUE\nDEV=TRUE" > .env
```

**Uso avanzado** - Con verificación:
```bash
# Después de los pasos básicos, verifica instalación
php -r "if(class_exists('Glory\Core\Setup')) echo 'Glory OK'; else echo 'Error';"
```

## Activar tema
Ve a WordPress Admin → Apariencia → Temas y activa "Template Glory".

¡Glory está listo para usar!
