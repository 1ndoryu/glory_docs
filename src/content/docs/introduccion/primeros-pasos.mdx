# Primeros Pasos con Glory

## Estructura básica
```
tu-tema/
├── Glory/          # Framework (agnóstico)
├── App/            # Tu código específico
│   ├── Config/     # Configuraciones
│   └── Functions/  # Funciones personalizadas
├── assets/         # CSS, JS, imágenes
└── functions.php   # Archivo principal
```

## Configuración inicial

**Uso mínimo** - Página básica:
```php
<?php
use Glory\Managers\PageManager;

// Crea página simple
PageManager::create('mi-pagina', [
    'title' => 'Inicio',
    'callback' => function() {
        echo '<h1>¡Glory funcionando!</h1>';
    }
]);
```

**Uso avanzado** - Con assets y opciones:
```php
<?php
use Glory\Managers\AssetManager;
use Glory\Managers\OpcionManager;

// Define assets automáticamente
AssetManager::defineFolder('assets/');

// Registra opción
OpcionManager::register('color_primario', [
    'valorDefault' => '#3498db',
    'tipo' => 'color',
    'etiqueta' => 'Color Principal'
]);

// Crea página con opción
PageManager::create('inicio', [
    'title' => 'Mi Sitio',
    'callback' => function() {
        $color = OpcionManager::get('color_primario');
        echo "<h1 style='color:$color'>¡Bienvenido!</h1>";
    }
]);
```
