# Cómo Usar Componentes

Glory incluye componentes reutilizables para acelerar el desarrollo. Todos los componentes son agnósticos y funcionan en temas y plugins.

## Componentes principales

**Uso mínimo** - Modal básico:
```php
<?php
use Glory\Components\Modal;
$modal = new Modal('mi-modal');
$modal->setTitle('Hola')->setContent('Contenido')->render();
echo '<button onclick="jQuery(\'#mi-modal\').show()">Abrir</button>';
```

**Uso avanzado** - Sistema completo:
```php
<?php
use Glory\Components\FormBuilder;

// Modal con formulario
$modal = new Modal('contacto-modal');
$modal->setContent(FormBuilder::campoTexto(['nombre'=>'email','label'=>'Email'])
                  . FormBuilder::botonEnviar(['accion'=>'enviar']));
$modal->render();
```

**Alertas** - Reemplazan alert() nativo:
```javascript
// Alerta simple
window.gloryAlert.show('Mensaje enviado', 'success');

// Con confirmación
window.gloryAlert.show('¿Eliminar?', 'warning', {
    onConfirm: () => eliminarItem()
});
```

**ContentRender** - Mostrar listas de contenido:
```php
<?php
use Glory\Components\ContentRender;

// Posts recientes
ContentRender::print('post', [
    'publicacionesPorPagina' => 5
]);
```

**FormBuilder** - Formularios sencillos:
```php
<?php
use Glory\Components\FormBuilder;

echo FormBuilder::inicio(['atributos' => ['id' => 'contacto']]);
echo FormBuilder::campoTexto(['nombre' => 'email', 'label' => 'Email']);
echo FormBuilder::botonEnviar(['accion' => 'enviar']);
echo FormBuilder::fin();
```

¡Usa estos componentes para acelerar tu desarrollo!
