## ¿Qué es AssetManager?

AssetManager gestiona scripts y estilos en Glory. Permite definir assets individuales o auto-descubrir carpetas completas.

## Uso básico

**Uso mínimo** - Asset individual:
```php
use Glory\Manager\AssetManager;

AssetManager::define('script', 'mi-script', '/assets/js/script.js', [
    'deps' => ['jquery'],
    'in_footer' => true
]);
```

**Uso avanzado** - Con localize y features:
```php
AssetManager::define('script', 'mi-script', '/assets/js/script.js', [
    'deps' => ['jquery'],
    'in_footer' => true,
    'area' => 'frontend',
    'localize' => [
        'nombreObjeto' => 'misDatos',
        'datos' => [
            'ajaxUrl' => admin_url('admin-ajax.php'),
            'nonce' => wp_create_nonce('mi_nonce')
        ]
    ],
    'feature' => 'miFeature' // Solo carga si feature está activa
]);
```

## Auto-descubrir carpeta

```php
// Define todos los CSS de una carpeta
AssetManager::defineFolder('style', '/assets/css/', [], 'tema-');
```

¡AssetManager simplifica la gestión de assets en WordPress!
