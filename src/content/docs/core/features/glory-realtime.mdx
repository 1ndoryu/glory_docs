## ¿Qué es `gloryRealtime`?

`gloryRealtime` permite actualizaciones en tiempo real mediante polling. Detecta cambios en el servidor y notifica al cliente para actualizar la UI.

## Uso básico

**Uso mínimo** - Escuchar cambios:
```php
use Glory\Services\EventBus;

// En servidor, emitir cambio
EventBus::emit('mi_canal');
```

```javascript
// En cliente, escuchar cambios
gloryRealtimePoll(['mi_canal'], { intervalMs: 3000 });

document.addEventListener('gloryRealtime:update', (e) => {
    if (e.detail.channel === 'mi_canal') {
        // Actualizar UI
        location.reload();
    }
});
```

**Uso avanzado** - Con actualización parcial:
```javascript
document.addEventListener('gloryRealtime:update', async (e) => {
    if (e.detail.channel === 'mi_canal') {
        const resp = await gloryAjax('obtener_datos_actualizados');
        if (resp.success) {
            document.querySelector('#contenido').innerHTML = resp.data.html;
        }
    }
});
```

¡Usa `gloryRealtime` para mantener la UI sincronizada!
